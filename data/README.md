# WELFake Dataset

**WELFake** is a large, balanced fake-news dataset widely used in machine learning and deep learning research on misinformation detection.  
It compiles articles from multiple open-source sources and applies unified preprocessing to provide a clean, reliable corpus.

---

## Dataset Summary

- **Total samples:** ~72,000  
- **Real news:** ~35,000  
- **Fake news:** ~37,000  
- **Balanced:** ✔ Yes (≈50/50 split)

---

## Columns

| Column | Description |
|--------|-------------|
| **title** | News headline |
| **text** | Full article content |
| **label** | `0 = Real`, `1 = Fake` |

---

## Why WELFake?

- ✔ **Balanced classes** → fairer model training  
- ✔ **Large-scale** → suitable for both ML & DL models  
- ✔ **Clean & standardized preprocessing**  
- ✔ **Diverse sources** → better generalization
